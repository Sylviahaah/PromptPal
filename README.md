## ğŸ“– Usage Guide

### Saving Prompts

#### Method 1: Right-Click Menu
1. **Select** any text on a webpage
2. **Right-click** to open the context menu
3. Choose **"Save to Prompt Library"**
4. (Optional) Add a title, category, and tags
5. Click **"Save"**

#### Method 2: Keyboard Shortcut
1. **Select** any text
2. Press `Alt+S`
3. Prompt saved with auto-generated title!

#### Method 3: Popup Quick Save
1. Click the **PromptPal icon** in toolbar
2. Paste or type your prompt in the text area
3. Click **"Save"**

### Inserting Prompts

#### On AI Chat Sites (ChatGPT, Claude, etc.)
1. Click in the chat input field
2. Press `Alt+P`
3. **Floating picker** appears with your prompts
4. Click any prompt â€” it's inserted automatically!

#### From the Popup
1. Click the **PromptPal icon**
2. Hover over any prompt
3. Click to copy or insert

### Managing Your Library

Access the full manager by:
- Clicking **"View All Prompts"** in the popup
- Right-clicking the icon â†’ **"Manage Prompt Library"**

In the manager, you can:
- **Search** â€” Find prompts by title, content, or tags
- **Filter** â€” View by category or tag
- **Sort** â€” By date, usage count, or title
- **Edit** â€” Modify title, content, category, tags
- **Delete** â€” Remove prompts you no longer need
- **Pin** â€” Keep favorites at the top

### Import & Export

#### Export (Backup)
1. Open **Settings** (âš™ï¸ icon)
2. Scroll to **"Data Management"**
3. Click **"Export Prompts"**
4. JSON file downloads to your computer

#### Import (Restore)

3. Select your backup JSON file
4. All prompts restored!

## â“ FAQ

<details>
<summary><strong>Is my data safe?</strong></summary>

Yes! All prompts are stored **locally in your browser** using Chrome's storage API. Your data never leaves your device and is not sent to any server.
</details>

<details>
<summary><strong>Does PromptPal work on [AI Platform X]?</strong></summary>

PromptPal uses smart detection to find input fields. It works with ChatGPT, Claude, Gemini, DeepSeek, Perplexity, and most other AI chat interfaces. If you find one that doesn't work, please [open an issue](#)!
</details>

<details>
<summary><strong>Can I sync prompts across devices?</strong></summary>

Currently, prompts are stored locally per browser. Cross-device sync is planned for a future premium feature. For now, use **Export/Import** to manually transfer your prompt library.
</details>

<details>
<summary><strong>What happens if I uninstall the extension?</strong></summary>

Your prompts are stored in Chrome's local storage. **Uninstalling will delete all saved prompts.** Always export a backup before uninstalling!
</details>

<details>
<summary><strong>Can I customize keyboard shortcuts?</strong></summary>

Yes! Go to `chrome://extensions/shortcuts` to change the default `Alt+S` and `Alt+P` shortcuts to any keys you prefer.
</details>

<details>
<summary><strong>Why does the extension need "Access all sites" permission?</strong></summary>

This allows PromptPal to:
- Save selected text from any webpage
- Insert prompts into any AI chat interface
- The extension only activates when you use the keyboard shortcuts or context menu. It does not track your browsing.
</details>
